### 提交规范

提交格式为"type: description"
- type: 提交类型
- description: 提交内容描述

其中提交类型具体描述如下：

| 提交类型     | 类型描述         |
|----------|--------------|
| revert   | 回退某个commit提交 |
| feat     | 新功能/新特性      |
| perf     | 功能、体验等方面的优化  |
| fix      | 修复问题         |
| style    | 样式相关修改       |
| docs     | 文档更新         |
| refactor | 代码重构         |
| chore    | 杂项修改         |
| ci       | 自动化流程配置或脚本修改 |
